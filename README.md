## Slack

### About

Chat with friends, colleagues and family wherever you are!

You could create new channels, rename and remove them

### New experience

* React
* Redux (including Redux Toolkit)
* Webpack
* Web Sockets
* Bootstrap React
* Deploy (Heroku)
* Cookies

### Needed technologies

* node v 15.8.0
* heroku cli

### Installing dependencies

```
npm install
```

### Starting

```
make start
```
